# Dandy's Portfolio Website

A modern developer portfolio built with **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**, containerized using **Docker** for consistent development and deployment environments.

## Live Demo

🌐 [View Portfolio](https://portfolio-dandh-hs-projects.vercel.app/)

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Docker
* Docker Compose
* Vercel

---

# Features

* Responsive modern UI
* Fast Next.js performance

* Mobile-friendly design

# Getting Started

## Clone the Repository

```bash
git clone https://github.com/dandh-h/portfolio.git
cd portfolio
```

---

# Run Locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---

# Docker Setup

This project is fully containerized using Docker and Docker Compose.

## Build and Run with Docker

```bash
docker compose up --build
```

Application will run on:

```text
http://localhost:3000
```

## Stop Containers

```bash
docker compose down
```

---

# Dockerized Environment

The application uses Docker to provide:

* Consistent development environments
* Simplified deployment workflow
* Dependency isolation
* Improved portability across systems

---

# Deployment

The portfolio is deployed using Vercel.

## Production Build

